This directory contains my work for creating a program that removes
the black edges of a image at the bit level as well as a program that can
test whether the image given represts a correctly solved sudoku puzzle where
the individual bits are the values 1-9 in the sudoku. These were created for
an assignment in Machine Structure and Assembly Language Programming at Tufts
University.
